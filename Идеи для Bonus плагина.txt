Файл *.ini в котором указываются бонусы для монеток
ID					: 	Бонуса, если не рандом :_D
ID монетки			: 	0 = Обычная монетка, 1 = Gold монетка, 2 = VIP монетка
Bonus 				:	Указываем id бонуса
Доп.Значение		:	Указывается тут дополительное значение в зависимости от бонуса
Кто может поднимать	:	0 - все, 1 - только зарегистрированные, 2 - Только VIP, 3 - Только Gold VIP
Вероятность выпаден	:	Имеется ввиду % выбора именно этого бонуса

; Бонусы:
; Shiling - Деньги на сайт ИСПОЛЬЗУЕТСЯ ТОЛЬКО при указании базы
; Money - Игровая валюта
; HP - Жизни
; weapon_ имя оружия, где кол-во пишем обоймы например в m4a1 90/30 = 3 значит пишем 3
; Gravity - Гравитация 
; Speed - Скорость 
; Kill - Убить игрока
; Damage - Нанести урон тому, кто подобрал
; Flags - дать флаг до конца карты (например VIP t)
; DoubleJump - Двойной прыжок
; HealthGrenade - Лечебная граната
; anew - выдаются очки anew
; exp - выдаётся опыт
